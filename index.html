¿Que es kali nethunter?

Es una plataforma de pruebas de penetración móvil gratuita y de código abierto para dispositivos Android, basada en Kali Linux . Para dispositivos rooteados que tienen una recuperación estándar (NetHunter Lite) y para dispositivos rooteados con recuperación personalizada para los que está disponible un kernel específico de NetHunter (NetHunter).Offensive Security publica imágenes oficiales en su página de descargas

Dispositivos compatibles:

NetHunter es compatible con una variedad de dispositivos Android. Las versiones más recientes amplían esta compatibilidad a dispositivos como el PinePhone, PinePhone Pro, Poco F1, OnePlus 6, OnePlus 6T y Xiaomi Mi MIX 2S.

Versiones mas populares

  • NetHunter Rootlessli
  • NetHunter Lite
  • NetHunter Full

Instalacion.

La instalación de NetHunter varía según la versión y el dispositivo. Para dispositivos no rooteados, es posible instalar NetHunter Rootless siguiendo guías específicas.

A continuaacion se presenta un link a un video que puede ayudar a entenderlo mas.

Características principales

Pruebas de seguridad

Realiza auditorías de seguridad y pruebas de penetración con herramientas avanzadas preinstaladas.

Compatibilidad HID

Emula dispositivos USB, como teclados, para realizar pruebas de inyección de teclas.

Soporte inalámbrico

Audita redes Wi-Fi con herramientas como Aircrack-ng y Reaver.

Requisitos

  • Dispositivo Android compatible.
  • Kernel personalizado con soporte NetHunter.
  • Acceso root para funciones avanzadas.

Usos comunes

Kali NetHunter se utiliza para:

  • Auditorías de redes inalámbricas.
  • Pruebas de aplicaciones móviles y web.
  • Evaluación de vulnerabilidades en sistemas.
  • Entrenamiento en ciberseguridad.
